#  Lightning Memory-mapped Database (LMDB) conan.io package

This is a [conan.io] package for [LMDB]. It is based on the [official fork] on GitHub and uses some changes from [ilyalukyanov/lightningdb-win].
Builds are currently targeted on Windows, but feel free to contibute.
 

[conan.io]: https://conan.io
[LMDB]: https://symas.com/products/lightning-memory-mapped-database/
[official fork]: https://github.com/LMDB/lmdb
[ilyalukyanov/lightningdb-win]: https://github.com/ilyalukyanov/lightningdb-win
